# Leela's Kitchen - Restaurant Website

A modern, responsive website for Leela's Kitchen featuring authentic Indian cuisine.

## Features

- Modern, responsive design using Next.js 14 and Tailwind CSS
- Menu page with extensive Indian cuisine offerings (Dosa, Idli, Biryani, and more)
- About page telling the restaurant's story
- Contact page with hours and contact information
- Mobile-friendly navigation

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser to see the website.

## Project Structure

```
├── app/
│   ├── layout.tsx          # Root layout with navbar and footer
│   ├── page.tsx            # Home page
│   ├── menu/page.tsx       # Menu page
│   ├── about/page.tsx      # About page
│   ├── contact/page.tsx    # Contact page
│   └── globals.css         # Global styles
├── components/
│   ├── Navbar.tsx          # Navigation component
│   └── MenuItem.tsx        # Menu item card component
└── public/                 # Static assets
```

## Technologies Used

- Next.js 14 (React framework)
- TypeScript
- Tailwind CSS
- Responsive design principles

## Build for Production

```bash
npm run build
npm start
```

## Customization

- Update contact information in `app/contact/page.tsx`
- Modify menu items in `app/menu/page.tsx`
- Edit restaurant story in `app/about/page.tsx`
- Customize colors in `tailwind.config.js`
